## 什么是免版权图片素材？为什么不能随便用网上找的图？

**免版权图片素材**，是指遵循 CC0（知识共享零协议）或公共领域许可的照片。这类图片的版权持有者已放弃全部权利，任何人都可以免费复制、修改、商用，无需署名，也不需要支付任何费用。

这和我们平时百度随手搜到的图完全不同。那些图大概率都有版权，拿来用很可能就是侵权。近几年图片版权纠纷案大幅增加，不少自媒体和企业都为此吃了罚单。

所以，找图要找**明确标注了 CC0 或公共领域**的来源。
